# Quickstart

Get your first modal up and running in just a few minutes. This guide will walk you through creating a basic modal with proper HTML structure and JavaScript initialization.

## Prerequisites

Make sure you have [installed Srfc](/installation) before continuing.

## Create a basic modal

<Steps>
  <Step title="Set up the HTML structure">
    Create a modal container with the `data-srfc` attribute and modal elements with unique IDs:

    ```html
    <!DOCTYPE html>
    <html lang="en">
    <head>
      <meta charset="UTF-8">
      <meta name="viewport" content="width=device-width, initial-scale=1.0">
      <title>My First Srfc Modal</title>
      <link rel="stylesheet" href="node_modules/srfc/dist/srfc.min.css">
    </head>
    <body>
      <!-- Modal trigger button -->
      <button data-srfc-open="welcome-modal">
        Open Modal
      </button>

      <!-- Modal container -->
      <div data-srfc>
        <!-- Modal element -->
        <div id="welcome-modal">
          <div class="modal-content">
            <h2>Welcome!</h2>
            <p>This is your first Srfc modal.</p>
            <button data-srfc-close>Close</button>
          </div>
        </div>
      </div>

      <script type="module" src="app.js"></script>
    </body>
    </html>
    ```

    <Note>
      The `data-srfc` attribute marks the container, `data-srfc-open` triggers opening, and `data-srfc-close` triggers closing.
    </Note>
  </Step>

  <Step title="Initialize Srfc">
    Create an `app.js` file and initialize Srfc:

    ```javascript
    import { Srfc } from 'srfc';
    import 'srfc/dist/srfc.min.css';

    const srfc = new Srfc();
    srfc.init();
    ```

    That's it! Your modal is now fully functional with:
    - Keyboard support (Escape to close)
    - Focus trapping
    - Backdrop click to close
    - Proper ARIA attributes
  </Step>

  <Step title="Add styling (optional)">
    Customize the modal appearance with CSS:

    ```css
    /* Basic modal styling */
    [data-srfc] {
      position: fixed;
      top: 0;
      left: 0;
      width: 100%;
      height: 100%;
      background: rgba(0, 0, 0, 0.5);
      display: none;
      align-items: center;
      justify-content: center;
    }

    [data-srfc].srfc-active {
      display: flex;
    }

    [data-srfc] > div {
      display: none;
    }

    [data-srfc] > div.srfc-active {
      display: block;
    }

    .modal-content {
      background: white;
      padding: 2rem;
      border-radius: 8px;
      max-width: 500px;
      width: 90%;
    }
    ```
  </Step>
</Steps>

## Programmatic control

You can also open and close modals programmatically:

<CodeGroup>
  ```javascript Open a modal
  import { Srfc } from 'srfc';

  const srfc = new Srfc();
  srfc.init();

  // Open a modal by ID
  srfc.open('welcome-modal');
  ```

  ```javascript Close a modal
  import { Srfc } from 'srfc';

  const srfc = new Srfc();
  srfc.init();

  // Close the currently active modal
  srfc.close();
  ```
</CodeGroup>

## Modal types

Srfc supports different modal types through the `data-srfc` attribute:

<Tabs>
  <Tab title="Basic">
    Default modal that can be dismissed with Escape or backdrop click:

    ```html
    <div data-srfc>
      <div id="basic-modal">
        <p>Basic modal content</p>
      </div>
    </div>
    ```
  </Tab>

  <Tab title="Linked">
    Modal tied to URL hash for shareable links and browser history:

    ```html
    <div data-srfc="linked">
      <div id="share-modal">
        <p>This modal's state is in the URL!</p>
      </div>
    </div>
    ```

    Access directly via: `https://yoursite.com#share-modal`
  </Tab>

  <Tab title="Arrow Navigation">
    Enable left/right arrow key navigation between sibling modals:

    ```html
    <div data-srfc="arrow">
      <div id="image-1">
        <img src="image1.jpg" alt="Image 1">
      </div>
      <div id="image-2">
        <img src="image2.jpg" alt="Image 2">
      </div>
      <div id="image-3">
        <img src="image3.jpg" alt="Image 3">
      </div>
    </div>
    ```
  </Tab>

  <Tab title="Strong (Non-dismissible)">
    Modal that cannot be dismissed with Escape or backdrop click:

    ```html
    <div data-srfc="strong">
      <div id="confirm-modal">
        <p>Are you sure? You must click a button to proceed.</p>
        <button data-srfc-close>Confirm</button>
      </div>
    </div>
    ```

    <Tip>
      Use `strong-backdrop` to only disable backdrop clicks, or `strong-esc` to only disable Escape key.
    </Tip>
  </Tab>
</Tabs>

## Listen to events

Srfc dispatches custom events when modals open and close:

```javascript
const modal = document.getElementById('welcome-modal');

modal.addEventListener('open.srfc', () => {
  console.log('Modal opened!');
});

modal.addEventListener('close.srfc', () => {
  console.log('Modal closed!');
});
```

## Complete example

Here's a complete working example combining everything:

<CodeGroup>
  ```html index.html
  <!DOCTYPE html>
  <html lang="en">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Srfc Example</title>
    <link rel="stylesheet" href="node_modules/srfc/dist/srfc.min.css">
    <link rel="stylesheet" href="styles.css">
  </head>
  <body>
    <h1>Srfc Modal Demo</h1>
    
    <button data-srfc-open="info-modal">Open Info Modal</button>
    <button data-srfc-open="confirm-modal">Open Confirm Modal</button>

    <!-- Basic modal -->
    <div data-srfc>
      <div id="info-modal">
        <div class="modal-content">
          <h2>Information</h2>
          <p>This is a basic modal with focus trapping.</p>
          <button data-srfc-close>Got it</button>
        </div>
      </div>
    </div>

    <!-- Strong modal -->
    <div data-srfc="strong">
      <div id="confirm-modal">
        <div class="modal-content">
          <h2>Confirm Action</h2>
          <p>This modal requires explicit confirmation.</p>
          <button data-srfc-close>Cancel</button>
          <button data-srfc-close>Confirm</button>
        </div>
      </div>
    </div>

    <script type="module" src="app.js"></script>
  </body>
  </html>
  ```

  ```javascript app.js
  import { Srfc } from 'srfc';
  import 'srfc/dist/srfc.min.css';

  const srfc = new Srfc();
  srfc.init();

  // Listen to modal events
  const infoModal = document.getElementById('info-modal');
  infoModal.addEventListener('open.srfc', () => {
    console.log('Info modal opened');
  });
  ```

  ```css styles.css
  [data-srfc] {
    position: fixed;
    top: 0;
    left: 0;
    width: 100%;
    height: 100%;
    background: rgba(0, 0, 0, 0.6);
    display: none;
    align-items: center;
    justify-content: center;
    z-index: 1000;
  }

  [data-srfc].srfc-active {
    display: flex;
  }

  [data-srfc] > div {
    display: none;
  }

  [data-srfc] > div.srfc-active {
    display: block;
  }

  .modal-content {
    background: white;
    padding: 2rem;
    border-radius: 8px;
    max-width: 500px;
    width: 90%;
    box-shadow: 0 4px 6px rgba(0, 0, 0, 0.1);
  }

  .modal-content h2 {
    margin-top: 0;
  }

  .modal-content button {
    margin-right: 0.5rem;
    padding: 0.5rem 1rem;
    cursor: pointer;
  }
  ```
</CodeGroup>

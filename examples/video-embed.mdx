title: Video Embed Modal
description: Modal with embedded YouTube or Vimeo video that auto-fits and resets on close
---

Embed videos in modals with automatic sizing and cleanup. The video stops playing when the modal closes, and the iframe is automatically fitted to its container while maintaining aspect ratio.

## How Embed Utilities Work

### The `data-srfc-embed` Attribute

Add this attribute to iframe elements to enable automatic handling:

```html
<iframe 
  data-srfc-embed="reset fit"
  width="1280" 
  height="720"
  src="https://www.youtube.com/embed/VIDEO_ID">
</iframe>
```

### Embed Types

You can use one or both of these space-separated values:

#### `reset`
Resets the iframe's `src` attribute when the modal closes, stopping video playback:

```javascript
// What happens internally:
const src = iframe.getAttribute('src');
iframe.setAttribute('src', src); // Reloads the iframe
```

#### `fit`
Automatically sizes the iframe to fit its parent container while maintaining aspect ratio:

```javascript
// Reads width and height attributes:
const ratio = height / width;

// Calculates size based on parent dimensions:
let w = parentWidth;
let h = w * ratio;
if (h > parentHeight) {
  h = parentHeight;
  w = h / ratio;
}

// Applies as inline styles:
iframe.style.width = w + 'px';
iframe.style.height = h + 'px';
```

<Note>
The `width` and `height` **attributes** (not style properties) are required for `fit` to work. They define the aspect ratio.
</Note>

## Why Reset Videos?

When a user closes a modal with a playing video:
- The video continues playing in the background (audio keeps playing)
- This creates a poor user experience
- The `reset` behavior stops playback by reloading the iframe

## Responsive Sizing

The `fit` behavior:
- Recalculates on window resize
- Maintains aspect ratio defined by attributes
- Never exceeds parent container dimensions
- Centers the video in the available space

## Common Video Embed URLs

### YouTube
```html
https://www.youtube.com/embed/VIDEO_ID
```

### Vimeo
```html
https://player.vimeo.com/video/VIDEO_ID
```

### With Autoplay
```html
https://www.youtube.com/embed/VIDEO_ID?autoplay=1
https://player.vimeo.com/video/VIDEO_ID?autoplay=1
```

<Tip>
Always include both `width` and `height` attributes on your iframe. These define the aspect ratio for the `fit` utility and are used by the browser even before JavaScript loads.
</Tip>

## Combining Features

You can combine video embeds with arrow navigation for a video gallery:

```html
<div data-srfc="arrow">
  <div id="video-1">
    <iframe data-srfc-embed="reset fit" width="1280" height="720" src="..."></iframe>
  </div>
  <div id="video-2">
    <iframe data-srfc-embed="reset fit" width="1280" height="720" src="..."></iframe>
  </div>
</div>
```

Users can navigate between videos using arrow keys, and each video will stop when they navigate away.

The simplest modal implementation requires a container with `data-srfc` attribute, modal elements with IDs, and buttons to open/close them.

## Complete Example

<CodeGroup>

```html HTML
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Simple Modal</title>
  <link rel="stylesheet" href="style.css">
</head>
<body>
  <!-- Open button -->
  <button data-srfc-open="welcome-modal">Open Modal</button>

  <!-- Modal container -->
  <div data-srfc="">
    <!-- Modal element -->
    <div id="welcome-modal">
      <div class="modal-content">
        <h2>Welcome!</h2>
        <p>This is a simple modal dialog.</p>
        <button data-srfc-close>Close</button>
      </div>
    </div>
  </div>

  <script src="srfc.js"></script>
  <script src="app.js"></script>
</body>
</html>
```

```css CSS
/* Container backdrop */
[data-srfc] {
  position: fixed;
  top: 0;
  left: 0;
  width: 100%;
  height: 100%;
  background: rgba(0, 0, 0, 0.5);
  display: none;
  align-items: center;
  justify-content: center;
  z-index: 1000;
}

/* Show container when active */
[data-srfc].srfc-active {
  display: flex;
}

/* Modal element */
[data-srfc] > * {
  opacity: 0;
  transform: scale(0.9);
  transition: opacity 0.3s, transform 0.3s;
}

/* Show modal when active */
[data-srfc] > .srfc-active {
  opacity: 1;
  transform: scale(1);
}

/* Modal content styling */
.modal-content {
  background: white;
  padding: 2rem;
  border-radius: 8px;
  max-width: 500px;
  width: 90%;
}

.modal-content h2 {
  margin-top: 0;
}

.modal-content button {
  padding: 0.5rem 1rem;
  background: #007bff;
  color: white;
  border: none;
  border-radius: 4px;
  cursor: pointer;
}

.modal-content button:hover {
  background: #0056b3;
}

/* Prevent body scroll when modal is active */
body.srfc-active {
  overflow: hidden;
}
```

```javascript JavaScript
// Initialize Srfc
const srfc = new Srfc();
srfc.init();
```

</CodeGroup>

## How It Works

1. **Container**: The `data-srfc=""` attribute marks the modal container (backdrop)
2. **Modal**: Each child element with an `id` becomes a modal
3. **Open Button**: `data-srfc-open="modal-id"` opens the specified modal
4. **Close Button**: `data-srfc-close` closes the currently active modal
5. **Initialization**: Call `srfc.init()` to register all modals and event listeners

## Behavior

When you click "Open Modal":
- The container gets the `srfc-active` class
- The modal element gets the `srfc-active` class
- Focus is trapped inside the modal
- Body scroll is disabled

The modal can be closed by:
- Clicking the close button
- Pressing the Escape key
- Clicking outside the modal (on the backdrop)

<Note>
The `srfc-active` class is automatically added by the library. Use it in your CSS to control visibility and animations.
</Note>
